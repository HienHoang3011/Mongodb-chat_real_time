📑 Index Strategy

1. Collection users - Indexes

idx_users_username (Unique) { username: 1 }
idx_users_email (Unique) { email: 1 }
idx_users_displayname_text (Text) { display_name: "text" }

2. Collection friendships - Indexes

idx_friendships_pair (Unique Compound) { requester_id: 1, recipient_id: 1 }
idx_friendships_recipient_status { recipient_id: 1, status: 1 }

3. Collection conversations - Indexes

idx_conversations_members { member_ids: 1 }
idx_conversations_updated { updated_at: -1 }
idx_conversations_type_members_br2 { type: 1, member_ids: 1 }

4. Collection conversation_members - Indexes

idx_convmembers_user_conv (Unique) { user_id: 1, conversation_id: 1 }
idx_convmembers_conv { conversation_id: 1 }

5. Collection messages - Indexes

idx_messages_conv_time { conversation_id: 1, created_at: -1 }
idx_messages_sender { sender_id: 1 }
idx_messages_content_text (Text) { content: "text" }

6. Collection attachments - Indexes

idx_attachments_uploader { uploader_id: 1 }

7. Collection notifications - Indexes

idx_notifications_recipient { recipient_id: 1, is_read: 1, created_at: -1 }
idx_notifications_ttl (TTL: 30 ngày) { created_at: 1 } (expireAfterSeconds: 2592000)
